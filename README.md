# ServiceOS Monorepo Scaffold

This repository starts as a minimal workspace for the ServiceOps platform.

## Layout

- `apps/` - product applications
- `packages/` - shared libraries and UI primitives
- `tooling/` - shared config packages

## Workspace

- Package manager: `pnpm`
- Task runner: `turbo`
